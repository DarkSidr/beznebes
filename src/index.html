Проблемы клинических исследований
Поиск пациентов
для клинических исследований
является уязвимым звеном процесса разработки новых лекарственных препаратов Половина проводимых клинических исследований не достигают поставленных целей по набору пациентов, а 11% из них не набирают ни одного пациента.
Спонсорам проблематично
подобрать клинический центр
отсутствует единая база центров из разных стран с оценками результатов работ по ранее проведенным исследованиям со стороны фармкомпаний, а с исследовательскими командами нет эффективного и надежного канала связи.
