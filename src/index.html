Преимущества
Наш продукт – это глобальная облачная технологическая платформа, включающая локализованные на разных языках web и мобильные приложения. Все они соединены с мощной серверной системой сбора и обработки данных по алгоритмам масштабируемой бизнес-логики с элементами искусственного интеллекта.
Основные модули системы – доступны в веб-версии и мобильном приложении:
Кабинет пациента с удобным поиском наиболее подходящего клинического исследования (получен патент) с функцией прескрининга для включения в выбранное исследование.
Кабинет исследователя с детальной информация о потенциальных исследованиях и пациентах, которые могут подходить для участия в исследовании.
Кабинет спонсора с возможностью подбора наиболее подходящих исследователей на основании полученной от них информации, анализа данных по набору в предыдущих исследованиях и прямой коммуникации с исследователями внутри системы.
Специализированный мессенджер для организации коммуникаций участников исследований и получения новостей по интересуемым темам по автоматически настраиваемому новостному каналу (получен патент).
Финансовая подсистема для организации платежей исследователям.
